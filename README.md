# Boilerplate

## Dependencies
* https://git-scm.com
* https://nodejs.org
* http://gruntjs.com
* http://bower.io/


## Install
```
#!bash
$ git clone https://github.com/kysela-petr/boilerplate.git
$ cd boilerplate
$ chmod +x install.sh
$ ./install.sh
$ grunt
```
### Open "dist/styleguide/index.html" in your browser.
